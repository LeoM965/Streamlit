# 🎮 Steam Analytics Pro

A premium Streamlit dashboard for exploring the Steam video games market. This project provides deep insights into game pricing, user satisfaction, and popularity trends using real-world data.

## 🚀 Features
- **Real-World Dataset**: Analyzes 1,000 top games from the Steam store.
- **Interactive Filtering**: Filter by multiple genres and price ranges.
- **Advanced Metrics**: Real-time calculation of Rating % and Review Volume.
- **Premium Visuals**:
    - **Price vs Quality**: Bubbles size based on popularity.
    - **Log-Scale Analysis**: Readable distribution of review volumes.
    - **Dark Mode UI**: Professional glassmorphism aesthetics.

## 🛠️ Installation & Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/LeoM965/Streamlit.git
   cd Streamlit
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the dashboard**:
   ```bash
   streamlit run dashboard.py
   ```

4. **Upload Data**: Use the sidebar to upload the provided `steam_games.csv` file.
